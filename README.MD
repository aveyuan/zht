### 关于
>这个项目用于记录我的妹妹“张红婷”的一些故事
 
 ### 时间节点
 - 2019年3月2日
 >今天早上，因为妹妹这几个月很不正常，妈妈今天刚好休假，便和妹妹一起去医院检查，早上我坐车回了老家，垫江，因为爷爷上个周生病住院了。
 回到老家的上午，妈妈便打电话过来告知，在“长安医院”CT检查出妹妹的脑部有积水，初步判断是脑部有肿瘤，检查后就就直接住院了。当天晚上我就赶着上来了。晚上到了家，我问妈妈，妹妹呢？妈妈说妹妹已经住院了，今天要去和妹妹一起。
 - 2019年3月3日
 >今天一大早就起来了，给妹妹买了早餐，带了牛奶，鸡蛋，妹妹睡在床上，她的手上输者液，状态不是很好，她说，她想吃玉米，我便直接回去煮玉米，回家也给妹妹带了一些常用的衣物，玉米煮好以后，送到了医院，妹妹也没怎么想吃了。
 下午，护士告诉我们，妹妹昨天晚上也吐了，出现了大脑颅内增压的情况，临时开了药，现在已经欠费了，让我们尽快去交费。下午妹妹给我说她想吃“拔丝蛋糕”，我便很快的去买了拔丝蛋糕，因为前面的CT无法更清楚的看到脑部实际情况，所以定在星期一做核磁共振。
 妈妈说，妹妹明天星期一学校还要照中考考试的照片，尽力看看能不能去照了。
 - 2019年3月4日
 >今天我去上班了，妈妈在照顾妹妹，妹妹在下午的时候做的核磁共振检查，报告要明天才能得到，抽了好多血，下班后去医院看妹妹，妹妹看上去有些虚弱了了，拔丝蛋糕差不多要吃完了，妹妹给我说，她还想吃，我说“好的，现在还有点，明天再给你们买”，妹妹说“谢谢哥哥”。
 - 2019年3月5日
 >因为担心长安医院的手上技术问题，妹妹在下午2点的时候转院到了“重庆医科大学附属第一医院”，下午3:30分，我还在公司上班，妈妈给我打电话，她那颤抖着哭泣着的声音告诉我，妹妹准备去手术了，现在头发都剃了，妈妈还在问哥哥呢，她还想着你的拔丝蛋糕。此时我也无法控制我的情绪，感觉全是都麻痹了，直接哭了出来。
 随后向公司请假。打车到五里店买了拔丝蛋糕，又前往重医1院，到了医院3楼，妹妹刚刚推进手术室，医生还在外面让爸妈签字。随后，开始了漫长的等待
 这次手术室做脑积水引流的手术，从下午5点正式开始，7:30结束，医生告诉我们，手术很成功，我们悬下的心终于落下了，随后在11楼重症监护室电梯口进行等待，7:50左右，我们看到了手术后的妹妹，她的意识还很清晰。眼睛是睁着的。

 - 2019年3月8日

 大概是晚10:30了，杨刚教授才到的办公室，她给我分析了妹妹肿瘤的病情，并表示，妹妹的这个肿瘤比他想象中的要严重，肿瘤长在大脑正中间，不规则的分散在了脑部颅咽管位置，他初步判断为颅咽管瘤，根据目前的情况，他说要尽快手术，积水下不去，切除肿瘤才是根本的办法，然后叫我们自己要做好心理准备，可能人财两失，这层楼还有17个人排着队等着切肿瘤。

 7:40左右，护工正式推着妹妹下去做手术，从此开始漫长的等待，晚上6点左右，手术出来了，妹妹是昏迷状态，由于妹妹去了重症监护室，2号床位也没了，晚上爸妈就收拾东西回去了。
